## [Operations with vectors]

### <ins>1. Main Calculations with Vectors — Cheatsheet</ins>
| Operation             | Notation | Component form                                       | Geometric meaning                   | Quick check / example         |
| --------------------- | -------- | ---------------------------------------------------- | ----------------------------------- | ----------------------------- |
| Zero vector           | **0**    | (0,…,0)                                              | Origin point                        | r + 0 = r                     |
| Additive inverse      | −r       | (−r₁,…,−rₙ)                                          | Same length, opposite direction     | r + (−r) = 0                  |
| Addition              | r + s    | (r₁+s₁,…,rₙ+sₙ)                                      | Head-to-tail                        | (3,2) + (−1,2) = (2,4)        |
| Subtraction           | r − s    | (r₁−s₁,…,rₙ−sₙ)                                      | Add the opposite of s               | r − s = r + (−s)              |
| Scalar multiplication | a·r      | (a·r₁,…,a·rₙ)                                        | Stretch/shrink; a<0 flips direction | 2·(3,2) = (6,4)               |
| Coordinates / basis   | r = B·c  | r = Σₖ (cₖ·bₖ) with B = [b₁ … bₙ] (basis in columns) | Decompose on a basis                | If B is invertible: c = B⁻¹·r |
